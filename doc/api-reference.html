    StoneBank keeps the contexts that define your domain and business logic.

    StoneBank.Accounts

    The Accounts context.

    StoneBank.Accounts.Encryption

    This module encrypt user password to save in database and verify if passed on login password is the same than stored.

    StoneBank.Accounts.User

    This scheme is a representation of the system user both admin and account owners. Only admin is allowed to see backoffice web pages.

    StoneBank.Products

    The Products context.

    StoneBank.Products.BankAccount

    This schema is a representation of bank account from user.

    StoneBank.Registers

    The Registers context.

    StoneBank.Registers.BankOperation

    This schema is a representation of bank operation made like deposit, withdraw and transfer.
